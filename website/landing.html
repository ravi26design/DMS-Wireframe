Features

Everything your team needs

Built ground-up for ecological field organisations with complex document workflows across multiple regions.

Smart Document Upload
Upload PDFs, Word, Excel, PPTX and more. Tag documents by theme, category, and visibility with a streamlined form.
Approval Workflows
Built-in multi-step approval with Pending, Approved, Request to Change, and Rejected states tracked per document.
Role-Based Access
Visitor, Registered User, Org Moderator, and Org Administrator roles — each with tailored views and permissions.
Org-Level Reports
Visual reports covering user contributions, document themes, file type distribution, and activity summaries across time periods.
Advanced Search
Filter by theme, category, file type, visibility, uploader, and date range. Global search with instant document suggestions.
Secure & Controlled
Public and Private visibility controls per document. Owner-based edit/delete rights ensure your data stays protected.
